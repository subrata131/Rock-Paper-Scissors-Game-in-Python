🚀 **Rock Paper Scissors Game in Python**

Just built a simple yet fun **Rock-Paper-Scissors CLI game** using Python! 🎮

### 🔥 Features:

* User vs Computer gameplay
* Random computer choices
* Score tracking (User | Computer | Draws)
* Continuous play until user exits
* Input validation for smooth experience

### 🧠 Concepts Used:

* Functions
* Loops (`while`)
* Conditional statements
* Dictionaries
* Random module

### 💻 How It Works:

* User inputs: `rock`, `paper`, or `scissor`
* Computer randomly selects its move
* Winner is decided using game logic
* Scores update after every round

### ▶️ Run the Program:

```bash
python r_p_s.py
```
